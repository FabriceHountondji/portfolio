---

## ⚙️ CI/CD et Déploiement Automatique

Ce projet utilise **GitHub Actions** pour un déploiement continu automatisé. À chaque push sur la branche `main`, le workflow `deploy.yml` :
1. Récupère le code source.
2. Installe les dépendances Node.js.
3. Construit le projet pour la production.
4. Déploie automatiquement le résultat sur **GitHub Pages**.

## 🚀 Installation Locale

Pour faire tourner ce projet sur votre machine, suivez ces étapes :

1. **Cloner le dépôt :**
   ```bash
   git clone https://github.com/FabriceHountondji/portfolio.git
   cd portfolio
   ```

2. **Installer les dépendances :**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement :**
   ```bash
   npm run dev
   ```
   *Le portfolio sera accessible à l'adresse `http://localhost:5173`.*
